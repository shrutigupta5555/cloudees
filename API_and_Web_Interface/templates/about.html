ABOUT

Cloudees is an application that lets users classify between clouds through such a simple and clean user interface. A user just need to select an image from the gallery, and then they can run the prediction and know what category does that specific cloud fall into. Under the hood, it works by utilising the powers of Deep Learning. Technically, it has been trained over 9000+ images of clouds of 9 major categories, through Convolutional Neural Networks, and has an accuracy of 90%. The request is sent from application to our hosted REST API, which resizes the image accordingly, and then performs the prediction and returns that to our user.
